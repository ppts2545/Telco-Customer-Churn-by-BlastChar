### บริษัทต้องการลดการยกเลิกบริการ (Churn) 
#### เราสวามารถ ไช้ churn ในการช่วยได้ เพราะ Data set ของบริษัทมีการเก็บข้อมูลที่จําเป็นครบถ้วนต่อการไช้ churn analysis เช่น (tenure, contract, usage, payment method) ซึ่งสามารถไช้ทํานายได้  และ รูปแบบธุรกิจโครงสร้างธุรกิจเหมาะสม และ churn จะสามารถช่วยตรงนี้ได้มาก เพื่อไห้ กิจการรักษา รายได้ subscribtion จากลูกค้า และ ปรับกลยุทธ์ เพื่อรักษา และ สอดคล้องกับการวิเคราะห์
